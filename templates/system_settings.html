Microsoft Entra ID Setup Instructions

  1. Go to Azure Portal
  2. Navigate to Entra ID > App registrations > New registration
  3. Set redirect URI to the value configured above (copy from the Redirect URI field)
  4. Copy the Application (client) ID and Tenant ID
  5. Go to Certificates & secrets > New client secret
  6. Copy the client secret value (available only once)
  7. Go to API permissions > Add a permission > Microsoft Graph > Delegated permissions > User.Read
⚠️ Important: The redirect URI must be configured in Azure before authentication will work.

Google OAuth Setup Instructions

  1. Go to Google Cloud Console
  2. Create a new project or select existing project
  3. Enable the Google+ API
  4. Go to Credentials > Create Credentials > OAuth client ID
  5. Set authorized redirect URI to: {{ request.url_root }}auth/oauth/google/callback
  6. Copy the Client ID and Client Secret
